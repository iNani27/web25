Introduction au Web

Comprendre les origines du Web, son lien avec Internet, et le rôle du W3C dans la standardisation.

1. Internet ≠ Web

Avant tout, Internet est l’infrastructure : un réseau mondial d’ordinateurs connectés.

Le Web (World Wide Web) est un service qui fonctionne sur Internet. Il permet de consulter des pages via un navigateur.

Le Web est un service qui utilise Internet pour afficher des pages via un navigateur.

2. Tim Berners-Lee : l'inventeur du Web

En 1989, Tim Berners-Lee travaille au CERN et cherche un moyen simple de partager des documents scientifiques.

Il crée :

  • HTML (HyperText Markup Language) : le langage pour structurer les pages web => pour structurer les pages
  • HTTP (HyperText Transfer Protocol) : le protocole de communication => pour transmettre les données
  • URL (Uniform Resource Locator) : l'adresse unique d’une ressource => pour localiser les ressources

WEB < Internet : le Web est une application qui fonctionne grâce à Internet.

3. W3C : le garant des standards

En 1994, Tim Berners-Lee fonde le W3C (World Wide Web Consortium).

Son rôle : définir les standards du Web pour assurer l’accessibilité, la compatibilité et l’ouverture des technologies web.
=> le Web reste ouvert, accessible, et compatible entre navigateurs.

En résumé

Tim Berners-Lee a inventé le Web avec HTML, sur Internet, et a fondé le W3C pour garantir son bon fonctionnement.
