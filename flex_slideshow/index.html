Vous êtes sur le page index.html, ce fichier est la racine de votre site web. Le dossier nommé flex_slideshow contient une structure complète du projet avec :

  • Fichier index.html
  • Dossiers css, js, images, audio
  • Slideshow fonctionnel (JS), inspiré de la section "How To" du site de réference en matière de standard du Web W3schools - How To SlideShow
  • Mise en page 100% Flexbox, inspiré de la section "CSS" du site de réference en matière de standard du Web W3schools - CSS FlexBox

Enivie de modifier les puces de cette liste ul? RDV dans le fichier style.css qui utilise la propriété et valeur list-style-type: cjk-ideographic;. Plus de style de liste sur W3schools : CSS list-style-type
