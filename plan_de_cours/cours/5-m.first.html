Principes de développement Mobile First

1. Objectifs principaux

  1. Supprimer la programmation superflue.
  2. Permettre un accès rapide aux informations.
  3. Supprimer les grandes images et les fonctions inutiles des pages.
  4. Développer avec un code source le plus léger possible.
  5. Abandonner le JavaScript pour développer les pages directement en HTML5.
  6. Créer des contenus clairs, faciles et rapides à comprendre.
  7. Créer des liens lisibles et faciles à cliquer.

2. Conception Mobile First

Défis à relever

  • Temps de chargement
  • Poids des éléments

Solutions

  • Amélioration progressive
  • CSS pour version mobile simplifiée, puis media queries pour versions plus grandes (responsive inversé)
  • Contenu média optimisé : images, vidéos (optimisées, taille max, min par défaut)
