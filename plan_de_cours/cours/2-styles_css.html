Plan du cours: HTML-CSS - partie 2

HTML pour le fond (robots de moteurs de recherche), CSS pour la forme (humain)

  • le HTML définit le fond (contenu, logique des éléments) ;
  • le CSS définit la forme (apparence).

Mettre en forme avec style !

  • couleur
  • police
  • taille du texte
  • bordure
  • fond (aplat ou dégradé de couleur; image)
  • ...
  • position et largeur des blocs de contenu

Historiquement

  1. Balises pour quelques styles : <font color="#aab1c3"> - 1991- - - ->
  2. 1996 : baliser chaque élément sur lesquels une mise en forme devait être appliquée => mise à jour, modification !!!
  3. Attribut style dans les balises (inline) : <p style="color:#aab1c3"></p>
  4. Déclaration des styles dans l'entête de chaque page (internal) 
    <head>
    ...
    <style>
        p {color: #aab1c3;}
    </style>
</head>
  5. Lien vers une/des feuilles de styles communes (external)
    ... Les feuilles de style (en cascade) étaient nées ... 
    <head>
    <link rel="stylesheet" href="mes_styles.css">
</head>

Les navigateurs web interprètent non seulement l'HTML mais aussi la CSS ! Sont-ils tous égaux dans cette entreprise? NON (cf.IE6).
Comme les langages WEB, les navigateurs ont eux-aussi leurs versions nées au fil du temps et plus la version du navigateur est ancienne, moins elle connait les fonctionnalités CSS.
OK pour le contenu - la page s'affichera toujours! Mais soyez structuré, si les styles ne sont pas interprétés, l'enchaînement des informations contenues doit rester cohérent.

Un ensemble de règles de styles se compose de:

  • un sélecteur (nom de la balise à laquelle je veux ajouter un style)
  • un bloc de déclaration de styles = une propriété : une valeur ;

Sélecteurs CSS

Types de sélecteurs :

  • l'élément (p, h1, ul, ...)
  • class (tous les élément qui ont la même class en attribut)
  • id (l'élément qui a un identifiant unique en attribut)

Démo

Les sélecteurs

  1. dans le fichier mes_liens.html, ajoutez la ligne de code le reliant à sa CSS
  2. Ajouter l'attribut class = "droite" sur 3 sélecteurs :
    • le premier H2
    • l'avant dernier paragraphe
    • le paragraphe contenant le lien haut de page
  3. créez un fichier nommé "styles_des_selecteurs.css"
  4. dans le fichier styles_des_selecteurs.css, ajoutez des styles:
  • Donnez des tailles différentes aux éléments de titres et paragraphes
  • Alignez à droite les éléments de class="droite"
  • Alignez au centre l'élément qui a l'ID top

Des propriétés CSS et leurs valeurs

Famille : Les couleurs

Propriété : Color (background-color, border-color, ...)

Valeur :

  • soit le nom de la couleur (en anglais) ;
  • soit le code hexadécimal de la couleur (ex: noir : #000000; blanc : #FFFFFF); ou encore
  • soit le code RGB (% de Red - de Green - de Blue) (ex: rouge :255,0,0 )
  • (CSS3 RGBa opacity)

Références en français ou en anglais

  • couleur => color
  • couleur de fond => background-color
  • background -image : https://www.w3schools.com/howto/howto_css_bg_change_scroll.asp
  • Liens effets : https://codepen.io/brenden/pen/RNZXqx et https://www.w3schools.com/css/css_link.asp

Unité, dimension, position

Unité

Valeur absolue : px

Valeur relative :

  • xx-small: minuscule ;
  • x-small: très petit ;
  • small: petit ;
  • medium: moyen ;
  • large: grand ;
  • x-large: très grand ;
  • xx-large: euh… gigantesque.
  • em (1.3em)
  • %

Responsive text

  • 1vw = 1% de la largeur de la fenêtre
  • 1VH = 1% de la hauteur de la fenêtre
  • 1vmin = 1vw ou 1VH, soit la valeur la plus petite
  • 1vmax = 1vw ou 1VH, soit la valeur la plus grande
