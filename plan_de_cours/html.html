🧱 Qu’est-ce que l'HTML ?

HTML signifie HyperText Markup Language. C'est le langage de base pour structurer une page web.

🔤 Structure minimale d’une page

<!DOCTYPE html>
<html>
  <head>
    <title>Titre de la page</title>
  </head>
  <body>
    <h1>Bonjour le monde</h1>
  </body>
</html>

📚 Balises utiles

  • <p> : paragraphe
  • <h1> à <h6> : titres
  • <a href=""> : lien
  • <img src=""> : image
  • <ul>, <ol>, <li> : listes
