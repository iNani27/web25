✨ Qu’est-ce que la CSS ?

CSS signifie Cascading Style Sheets. C’est le langage utilisé pour styliser l’apparence d’un site web.

📌 Exemple simple

h1 {
  color: deeppink;
  text-align: center;
}

🔍 Sélecteurs de base

  • h1 : toutes les balises h1
  • .classe : tous les éléments avec la classe "classe"
  • #id : l’élément avec l’ID "id"

🎨 Propriétés courantes

  • color : couleur du texte
  • background-color : couleur de fond
  • padding / margin : espace intérieur / extérieur
  • font-size / font-family : taille / police
